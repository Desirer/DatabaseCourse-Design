英语学习与测试平台
===============

## 主要功能

（1）公共模块

登录注册、个人信息修改、密码修改

（2）教师模块

文章查看发布、班级管理、测试查看发布批改、历史成绩查看

（3）学生模块

文章查看、参与测试、查看成绩

（4）信息管理模块

学生信息管理、教师信息管理（待完善）、文章管理（待完善）、考试管理（待完善）、成绩管理（待完善）
